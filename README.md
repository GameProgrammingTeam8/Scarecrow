# 🎮 게임 소개
- 모바일 앱으로 즐길 수 있는 3D Top-View 핵앤슬래시 던전 탈출 게임입니다.
- 튜토리얼 맵에서 조작법을 익힐 수 있습니다.
- 던전에서 몰려오는 적들을 죽이거나 피하면서 던전 곳곳에 배치된 트리거 허수아비를 모두 파괴하면 탈출을 위한 문이 열리고, 무사히 탈출하면 승리하는 게임입니다.
